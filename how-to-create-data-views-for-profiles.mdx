## Overview

Data Views give you a simplified version of the **All User Data** tab so you only see the profile fields that matter for your work. Each Data View is a subset of all available fields, arranged in the order you choose.

Last Updated: **February 3, 2026**

<Callout kind="info">

Data Views are specific to your login. Any Data View you create or edit is visible only to you and does not affect what other staff members see.

</Callout>

## When to use a Data View

Use a Data View when you:

- Spend time scrolling through the All User Data tab to find a few key fields.
- Want a different field layout than other staff members without changing their view.
- Need different field sets for different user types, such as campers, parents, or staff.

Once you set up a Data View, you can switch to it from the **Data Views** tab on any profile of the matching user type.

## Create a new Data View

Follow these steps to create a new Data View from any profile.

<Steps>
  <Step title="Open a profile and go to the Data Views tab" icon="users" title-type="p">

  - Go to the relevant user's profile.
  - Click the **Data Views** tab on the profile.

  </Step>

  <Step title="Open Data View settings" icon="settings" title-type="p">

  - On the **Data Views** tab, click the **gear** icon to open Data View settings.

  </Step>

  <Step title="Choose a user type and add a new view" icon="plus" title-type="p">

  - From the settings screen, choose the **user type** the view should apply to, such as camper, parent, or staff.
  - Click **Add New Data View**.
  - Enter a clear name for the view if prompted and save your changes.

  </Step>

  <Step title="Edit fields for your Data View" icon="edit" title-type="p">

  - In the list of Data Views, find the one you just created.
  - Click the **icon to the right** of the Data View name.
  - Select **Edit Fields** to open the field selection popup.

  </Step>

  <Step title="Select and add profile fields" icon="code" title-type="p">

  - In the popup, browse or search for the profile fields you want in this view.
  - Select each field you want to include.
  - Click **Add** to add the selected fields into the view.

  </Step>

  <Step title="Reorder fields by drag and drop" icon="folder" title-type="p">

  - After you add fields, drag and drop them into the order you prefer.
  - Move the most important fields to the top so they are visible without scrolling.

  </Step>

  <Step title="Use your new Data View on profiles" icon="play-circle" title-type="p">

  - Return to the user's profile.
  - Open the **Data Views** tab.
  - From the Data View selector, choose the Data View you just created.
  - Confirm that only the selected fields appear, in the order you set.

  </Step>
</Steps>

## Verify and adjust your Data View

After you apply the new Data View, review it on a few different profiles of the same user type to make sure it works as expected.

If you need to change the fields or their order:

- Go back to the **Data Views** tab.
- Click the **gear** icon.
- Locate your Data View and use **Edit Fields** again to update the configuration.

Your changes take effect the next time you select that Data View on a profile.
