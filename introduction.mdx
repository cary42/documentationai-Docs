## Overview

Campium empowers camps of every size to run smarter operations. You get a connected platform to manage registration, billing, attendance, transportation, medication, and communication. Turn day-to-day tasks into organized, data-driven decisions that help your camp thrive.

With Campium, you centralize all camp management in one place. Access custom reporting, mobile rostering, and secure camper profiles. Whether you run a small day camp or a large overnight program, Campium scales to meet your needs.

## Integrate with Our API

For advanced automation, use the Campium API.

<CodeGroup tabs="JavaScript,Python">
````javascript
const response = await fetch('https://api.example.com/v1/campers', {
  headers: {
    'Authorization': `Bearer ${YOUR_API_KEY}`
  }
});
const campers = await response.json();
console.log(campers);
````
````python
import requests

response = requests.get(
    'https://api.example.com/v1/campers',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
)
campers = response.json()
print(campers)
````
